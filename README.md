# Dot-matrix-Printer-Using-Arduino-
The system interprets a bitmap array input to very closely replicate images or patterns
on paper , implemented using Arduino Uno. The plotter features a pen driven by a
micro-servo for vertical Z-axis motion and a belt-and-pulley system for X-axis
movement of pen carriage. The Y-axis utilizes dual motors with rubber wheels for
paper traction
